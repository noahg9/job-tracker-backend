## Tech Stack

- **Backend:** .NET 8 Web API

---

## Getting Started

### Prerequisites

- [.NET 8 SDK](https://dotnet.microsoft.com/download)

### Local Setup

1. Clone the repo
2. Run backend:
   ```bash
   cd JobApplicationTracker.Api
   dotnet run
	```
